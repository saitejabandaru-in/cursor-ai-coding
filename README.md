## What Is Cursor?

Cursor is VS Code — but with AI deeply integrated into every part of the editing experience.

## Cursor Modes

### Chat (Cmd+L)
Ask questions about your code without editing.
Use @file, @codebase, @docs, @web for context.

### Composer (Cmd+I)
Edit code across multiple files with natural language instructions.

### Agent Mode
Autonomous coding — reads files, writes code, runs terminal commands.
Use for feature development, refactoring, and bug fixing.

## .cursorrules

Create `.cursorrules` in your project root to set:
- Your preferred stack and libraries
- Code standards (PEP 8, type hints, etc.)
- Things to always/never do
- Project architecture overview

## Power Workflows

### TDD with Cursor
1. Write tests describing desired behavior
2. Cursor Agent implements the functions
3. Run tests — Cursor fixes failures
4. Human reviews the implementation

### Refactoring Legacy Code
Use @codebase to give Cursor full project context,
then describe the refactoring goal incrementally.
